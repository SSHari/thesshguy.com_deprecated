# Clamp Demos

The following are demos for the css `clamp` utility.

## Max Width Clamp

The following styles are applied to the demo. The clamped class allows for the mixed use of the `max-width` property which constrains the element's max width to `900px` **or** `100%`.

<CodeBlock lang="css" code={maxWidthClampDemo} />

<div className="overflow-hidden resize-x max-w-full">
  <p className="text-center">
    Resize the container to see how the two are affected.
  </p>
  <div
    style={{ minWidth: '600px', width: '50%', maxWidth: '900px' }}
    class="border-solid border border-gray-900 rounded my-8 mx-auto p-4 text-center"
  >
    Not clamped
  </div>
  <div
    style={{ width: 'clamp(600px, 50%, 900px)', maxWidth: '100%' }}
    class="border-solid border border-gray-900 rounded my-8 mx-auto p-4 text-center"
  >
    Clamped
  </div>
</div>
